## swap fileを追加 (ubuntu)

現在の状態の確認
```
sudo swapon -s
```
swap file作成して割り当て
```
sudo dd if=/dev/zero of=/swapfile bs=2M count=2048
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
```
`sudo swapon -s`で確認  
`free`でも確認できる

再起動時にも有効になるようにするため、/etc/fstabに以下を追加する
```
/swapfile none swap sw 0 0
```
